# ContactsManager

## Description

Petit projet Java pour gérer des contacts.

Le programme permet :

- d’ajouter des contacts
- de rechercher un contact
- d’afficher un numéro de téléphone

---

## Classes

### Contact
Contient :
- le nom
- le numéro de téléphone

### ContactsManager
Permet :
- d’ajouter des contacts
- de rechercher des contacts

### Main
Classe principale pour tester le programme.

Contacts utilisés :
- Jean
- Marc
- Foto
- Audilonne
- Cheick

---

## Exemple

```text
Numéro de Cheick : 0909090909
```

- Java
